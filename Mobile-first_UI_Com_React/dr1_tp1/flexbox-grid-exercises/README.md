# Exercícios de Flexbox e Grid Layout

Projeto com 16 exercícios práticos de CSS Flexbox e Grid Layout em React.

## Como usar

1. Instale as dependências:
```bash
npm install
```

2. Execute o projeto:
```bash
npm start
```

3. Acesse: `http://localhost:3000`

## Exercícios incluídos

### Flexbox (1-13)
- Flex-direction (row-reverse, column-reverse)
- Flex-grow, flex-shrink, flex-basis  
- Justify-content (flex-start, flex-end, center, space-between, space-around)
- Align-items (centralização completa)
- Containers aninhados
- Layouts responsivos

### Grid Layout (14-16)
- Grid básico (4 colunas x 3 linhas)
- Grid com spans (itens ocupando múltiplas células)
- Colunas proporcionais (1fr, 2fr, 1fr)

## Tecnologias

- React 18
- React Router
- CSS3 (Flexbox & Grid)

## Estrutura

```
src/
├── components/
│   ├── Home.js              # Lista de exercícios
│   ├── BackButton.js        # Botão voltar
│   └── exercises/           # 16 exercícios
├── App.js                   # Rotas
└── App.css                  # Estilos globais
```

Cada exercício mostra o resultado visual e o código CSS aplicado.
