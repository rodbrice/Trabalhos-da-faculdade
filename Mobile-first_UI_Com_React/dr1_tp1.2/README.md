# React Native TP1.2 - 16 Tarefas 🚀

Projeto completo com 16 tarefas de React Native desenvolvidas para o TP1.2 da faculdade.

---

## 📱 Como Testar no Snack Expo

### Opção 1: Importar do GitHub (Recomendado)

1. Acesse: **https://snack.expo.dev**
2. Clique em **"Import git repository"** (canto superior esquerdo)
3. Preencha os campos:
   - **Repository URL**: `https://github.com/rodbrice/Trabalhos-da-faculdade`
   - **Folder path**: `Mobile-first_UI_Com_React/dr1_tp1.2/react-native-tp`
   - **Branch name**: `master`
4. Clique em **"Import"**
5. Aguarde o carregamento e teste no simulador ou escaneie o QR Code com o app Expo Go

## ✅ Tarefas Implementadas

### 📦 Componentes Básicos
- **Tarefa 1**: Tela de boas-vindas com View e Text
- **Tarefa 2**: Login básico com validação de campos
- **Tarefa 3**: Image com descrição ajustável
- **Tarefa 4**: ScrollView com lista de 8 itens
- **Tarefa 5**: Duas imagens lado a lado (10px de espaçamento)

### 🎨 Interatividade com Botões
- **Tarefa 6**: Três botões personalizados (cores, tamanhos e bordas)
- **Tarefa 7**: Botões com Alert personalizado
- **Tarefa 8**: TouchableOpacity com interação e Alert

### 📋 Layouts e Listas
- **Tarefa 9**: Tela de perfil com foto, nome, biografia e estatísticas
- **Tarefa 10**: Lista de compras com FlatList
- **Tarefa 11**: Galeria horizontal com ScrollView

### 🔢 Estados e Interações
- **Tarefa 12**: Contador de cliques simples
- **Tarefa 13**: Contador V2 (incrementar/decrementar sem negativos)
- **Tarefa 14**: Pressable com mudança de cor aleatória

### 📰 Componentes Avançados
- **Tarefa 15**: Página de notícias com cards estilizados
- **Tarefa 16**: Campo de feedback multiline com limite de caracteres
