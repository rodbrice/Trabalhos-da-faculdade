# Sistema de Reserva de Passagens de Ônibus

## Expresso Horizon

Sistema completo de gestão de reservas de passagens de ônibus desenvolvido em React + TypeScript.

### Tecnologias

- React 18.2
- TypeScript 4.9
- CSS Puro (sem frameworks)

### Funcionalidades

1. ✅ Cadastro de novas reservas
2. ✅ Validação de assentos duplicados (1-40)
3. ✅ Edição de reservas existentes
4. ✅ Cancelamento de reservas
5. ✅ Busca por destino em tempo real
6. ✅ Feedback visual com mensagens temporárias
7. ✅ Interface responsiva (mobile-first)
8. ✅ Tabela com status coloridos
9. ✅ Validação completa de formulários
10. ✅ Design moderno com gradiente roxo

### Como executar

```bash
npm install
npm start
```

O projeto será aberto em [http://localhost:3000](http://localhost:3000)
