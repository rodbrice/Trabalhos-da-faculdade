# Trabalhos da Faculdade

Repositório com projetos práticos de desenvolvimento front-end.

## Projetos

**Desenvolvimento Front-end com Frameworks**
- **React - 16 exercícios** (`Fundamentos_React/dr1_tp1/react-exercises-app`)
  - Aplicação React 19 com exercícios práticos
  - Para executar: `npm install` → `npm start`

**Mobile-first UI com React**
- **Flexbox & Grid** (`Mobile-first_UI_Com_React/dr1_tp1/flexbox-grid-exercises`)
  - Exercícios de layout responsivo

## Como usar

```bash
git clone https://github.com/rodbrice/Trabalhos-da-faculdade.git
cd Trabalhos-da-faculdade

# Para o projeto React:
cd Desenvolvimento_Front-end_com_Frameworks/Fundamentos_React/dr1_tp1/react-exercises-app
npm install
npm start
```

## Deploy

O projeto React pode ser acessado online via Vercel (em configuração).
